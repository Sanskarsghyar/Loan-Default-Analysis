# Loan-Status-Analysis
- Encountered with the features responsible for clearing or not clearing the loan
- Applied Classification technique to build Logistic Regression Model to predict the loan status
- Compared and analysed the predicted and actual loan status using Confusion Matrix
• Analysed the driver features responsible while granting loans, leading to a deeper understanding of the risk of defaults
• Employed Logistic Regression, Decision Tree Classifier and Random Tree Classifier models for predicting the loan status
• Evaluated the performance of the model from accuracy and f1score of predicted loan status from Confusion Matrix
